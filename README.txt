Introduction
============

This package provides Ushahidi-like maps for Plone CMS.


Usage
~~~~~

Package provides @@ushahidi-map-view view to be used on site root or any other
folderish object. It gets all GEO aware nested objects and shows them on a map.
There you can also filter those objects out by keywords, portal type and dates.
